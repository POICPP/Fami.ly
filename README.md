# fami.ly – The Private Family Ride-Sharing App

Welcome to fami.ly. This is a private app just for families. It replaces annoying text messages like "Where are you?" or "Can you pick me up?" with a simple map interface.

## How it works
Instead of paying with money, you pay with favors. If someone drives you, you owe them something in return, like emptying the dishwasher or mowing the lawn. It keeps things fair without involving cash.

## What is in this project
This code includes a landing page and a working prototype of the app. You can simply open the HTML file in your browser to try it out. 

Inside the prototype, you can log in as **Max** or **Michael** using the password **123**. 
* **Max** is the passenger who taps on the map to request a ride. 
* **Michael** is the driver who receives the request and sees the green route. 

The app automatically calculates the distance and even tells you if it is faster to just walk.

## Tech stuff
It is built with basic HTML, CSS, and JavaScript. The maps are powered by Leaflet and OpenStreetMap. No commercial tools, just a fun and simple way to organize family logistics.
